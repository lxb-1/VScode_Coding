### 如何将VScode项目上传到github仓库
在VScode下面的的bash终端如下如所示：
![](../Figures/VScode%E7%BB%88%E7%AB%AF.png)
输入如下几个命令就可以实现将VScode中的代码推送到自己的Github仓库中了。
```bash
git add .               # 将项目下的所有文件放入暂存区
git commit -m "note"    # 提交到本地仓库
git push                # 提交到远程仓库github上
```
